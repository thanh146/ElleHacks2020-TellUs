Tell Us Web Application

Operating System Support: macOS Mojave or more 

Require:
	Node.js
	AngularJS library
	
Running the app:
	1. Open Terminal
	2. Navigate to the project folder
	3. Type 'nodemon index' to run the server
	4. Open any web browser
	5. Type 'http://localhost:4000/home.html' into the address bar
